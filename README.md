In Typescript, we can use common object-oriented patterns.
- Classes
- Interfaces
## Classes
#### Class Members
  Here Is the Most basic class - an empty one: 
```
class Point {}
```
#### Fields
##### Field declaration
```
class Point {
    x: number;
    y: number;
}
const pt = new Point();
pt.x = 0;
pt.y = 1;
```
##### Field Initialization
```
const Point {
    x = 0;
    y = 1;
}
const pt = new Point();
consolel.log(pt.x, pt.y)
```
